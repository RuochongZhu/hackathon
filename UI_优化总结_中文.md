## 💡 使用建议

### 添加新组件时
1. 使用设计系统变量 (间距、字号、颜色)
2. 遵循 8px 网格对齐
3. 添加悬停效果和过渡
4. 确保响应式兼容

### 修改样式时
1. 在 `dashboard/styles.py` 中修改
2. 使用 CSS 变量而非硬编码
3. 保持命名一致性
4. 添加注释说明

### 添加图标时
1. 在 [Lucide Icons](https://lucide.dev) 查找
2. 使用 `icon()` 函数创建
3. 指定合适的尺寸和样式类
4. 测试显示效果

---

## 🔧 常见问题

### 图标不显示
**解决**: 检查 Lucide CDN 加载，在控制台运行 `lucide.createIcons()`

### 样式不生效
**解决**: 检查 `CUSTOM_CSS` 加载，清除浏览器缓存

### 动画卡顿
**解决**: 确保使用 `transform` 而非 `left/top`，检查硬件加速
