# UI 优化测试指南

## 启动应用

### 方法 1: 使用 Shiny
```bash
cd /Users/zhuricardo/Desktop/hackathon
shiny run dashboard/ui_app.py --reload --port 8001
```

### 方法 2: 使用 Python
```bash
cd /Users/zhuricardo/Desktop/hackathon
python -m shiny run dashboard/ui_app.py --port 8001
```

然后在浏览器中访问: http://localhost:8001
